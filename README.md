# visual-python

> Visual Python

## Running tests

### On Windows

```shell
> py -3 -m unittest
```

### On GNU/Linux

```shell
$ python3 -m unittest
```

### On Mac OS X

```shell
$ python3 -m unittest
```

## Running app

### On Windows

```shell
> py -3 -m visual_python
```

### On GNU/Linux

```shell
$ python3 -m visual_python
```

### On Mac OS X

```shell
$ python3 -m visual_python
```
