# visual-python

> Visual Python

## Running tests

### On Windows

```shell
> py -3 -m unittest
```

### On GNU/Linux

```shell
$ python3 -m unittest
```

### On Mac OS X

```shell
$ python3 -m unittest
```

## Running

### On Windows

```shell
> py -3 -m __init__
# or
> py -3 __init__.py
```

### On GNU/Linux

```shell
$ python3 -m __init__
# or
$ python3 __init__.py
```

### On Mac OS X

```shell
$ python3 -m __init__
# or
$ python3 __init__.py
```
